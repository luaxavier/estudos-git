## Principais comandos

#### Inicializando repositorio
```
git init
```

#### Verificando status dos arquivos
```
git status
```

#### Adicionando Arquivos ao stage
```
git add nome-do-arquivo.txt
git add nome-*
git add *-arquivo.txt               
git add .
```

#### Comparando as mudanças feitas no arquivo
```
git diff
```
